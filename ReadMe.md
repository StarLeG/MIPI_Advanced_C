# Учебный проект: C-приложения и игра "Змейка"

## Описание проекта
Проект представляет собой набор учебных C-приложений для изучения различных аспектов программирования на языке C, включая работу с битами, массивами, структурами данных, а также классическую игру "Змейка" с использованием библиотеки ncurses.

### Содержимое проекта:
- **Test_A00-A10**: Серия учебных программ для практики работы с базовыми и продвинутыми концепциями C
- **Snake**: Классическая игра "Змейка" с текстовым интерфейсом

## Требования к инструментам

### Основные инструменты:
1. **WSL (Windows Subsystem for Linux)**
   - Рекомендуется Ubuntu 20.04/22.04
   - Установка через Microsoft Store или PowerShell

2. **Visual Studio Code**
   - С поддержкой удаленной разработки
   - Расширения для работы с WSL

3. **Компилятор C и система сборки**
   - GCC/G++ компилятор
   - CMake (версия 3.28.3 или выше)
   - Make

## Установка инструментов

### Шаг 1: Установка WSL
```powershell
# Откройте PowerShell от имени администратора и выполните:
wsl --install
```

### Шаг 2: Установка необходимых пакетов в WSL
```bash
# Обновление списка пакетов
sudo apt update && sudo apt upgrade -y

# Установка компилятора и утилит сборки
sudo apt install -y gcc g++ cmake make

# Установка библиотеки ncurses (для игры "Змейка")
sudo apt install -y libncurses5-dev libncursesw5-dev
```

### Шаг 3: Установка Visual Studio Code
1. Скачайте и установите VS Code с официального сайта
2. Установите расширение "Remote - WSL"

### Шаг 4: Настройка проекта в VS Code
1. Откройте VS Code
2. Нажмите F1 и выберите "Remote-WSL: New Window"
3. Откройте папку проекта в WSL
4. При необходимости установите расширение для C/C++

## Сборка проекта

### Автоматическая сборка (рекомендуется)
```bash
# Дайте права на выполнение скрипта
chmod +x Build.sh

# Запустите скрипт сборки
./Build.sh
```

### Ручная сборка
```bash
# Создание директории для сборки
mkdir -p build
cd build

# Генерация файлов сборки
cmake ..

# Компиляция проекта
make
```

## Запуск приложений

### Автоматический запуск
```bash
# Дайте права на выполнение скрипта
chmod +x Run.sh

# Запустите скрипт
./Run.sh
```

### Ручной запуск отдельных приложений
```bash
# Запуск тестовых приложений (пример)
./build/apps/Test_A/A00/test_A00
./build/apps/Test_A/A01/test_A01
# ... и так далее для других приложений

# Запуск игры "Змейка"
./build/apps/snake/snake
```

## Решение возможных проблем

### Ошибка: "ncurses not found"
```bash
# Установите библиотеку ncurses
sudo apt install libncurses5-dev libncursesw5-dev
```

### Ошибка: "CMake version is too old"
```bash
# Обновите CMake
sudo apt remove cmake
sudo apt install -y software-properties-common
sudo add-apt-repository -y ppa:george-edison55/cmake-3.x
sudo apt update
sudo apt install -y cmake
```

### Ошибка прав доступа
```bash
# Дайте права на выполнение скриптов
chmod +x Build.sh Run.sh
```

## Дополнительные возможности

### Отладка в VS Code
1. Откройте файл .c в редакторе
2. Установите точку останова (F9)
3. Запустите отладку (F5)
4. Выберите конфигурацию "C/C++: gcc build and debug active file"

### Сборка конкретного приложения
```bash
cd build
make test_A00  # Собрать только приложение A00
make snake     # Собрать только игру "Змейка"
```
